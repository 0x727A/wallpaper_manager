# Wallhaven 图片裁剪记录器

一个桌面应用，用于批量管理壁纸图库的裁剪记录。支持可视化框选裁剪、批量处理、跳过标记、已裁剪图片管理。

## 下载安装

从 [Releases](../../releases) 页面下载对应平台的安装包：

- **macOS**：下载 `.dmg` 安装包，拖拽到 Applications
- **Windows**：下载 `.msi` 安装包，双击安装

> **macOS 首次打开时若提示「无法验证开发者」或「已损坏」**，请按住 Control 键右键点击应用图标选择「打开」。如果仍不行，在终端执行：
> ```bash
> xattr -dr com.apple.quarantine /Applications/Wallhaven\ 图片裁剪记录器.app
> ```

## 功能

- **图库扫描**：扫描本地图库目录，支持文件夹分类，自动识别 NSFW 内容
- **裁剪编辑器**：可视化框选裁剪区域，支持 8 种常用比例（16:9、4:3、1:1 等），滚轮缩放，实时预览
- **裁剪记录**：每张原图可保存多条裁剪记录，数据自动保存在图库根目录的 `crops.json`
- **状态筛选**：左侧图库支持按「全部 / 已裁剪 / 未裁剪 / 跳过」筛选
- **跳过标记**：对不打算裁剪的图片标记跳过，写入 `skipped.json`，可随时取消
- **已裁剪画廊**：网格展示所有已裁剪图片，支持大图预览、键盘左右导航、重新裁剪、删除已裁图片
- **批量裁剪**：导入 `crops.json` 即可按已有记录自动批量裁剪
- **原图回收**：删除原图时移动到图库内 `_deleted` 目录，可手动恢复
- **深色模式**：自动跟随系统深浅色主题

## 快速上手

1. **选择图库目录**：点击顶部「图库」旁的文件夹按钮，选择你的壁纸根目录
2. **选择输出目录**：点击「输出」旁的文件夹按钮，选择裁剪图保存位置（默认在图库同级目录）
3. **扫描**：点击「扫描」加载图片列表
4. **裁剪**：左侧点击一张图，在编辑器中框选区域，点击「保存裁剪」
5. **跳过**：不想裁剪的图，点击「跳过」标记
6. **查看已裁剪**：点击顶部「显示已裁剪」开关或左侧筛选下拉框切换视图

## 数据文件

应用会在图库根目录自动生成以下文件：

| 文件 | 说明 |
|:---|:---|
| `crops.json` | 所有裁剪记录，可导入导出 |
| `skipped.json` | 被标记跳过的图片记录 |
| `_deleted/` | 被删除原图的回收目录 |
